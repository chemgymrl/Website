ChemGymRL Benches

The ChemGymRL environment has four benches available: the reaction bench, the extraction bench, the distillation bench and the characterization bench. The reaction bench performs a series of reactions in a single vessel, with an agent choosing from a select family of reactions to perform on the materials in the vessel. The extraction bench moves materials between a series of vessels by separating them using the different densities of the materials in the initial vessel, with the help of the agent selecting an appropriate solute to add into the mixture to separate the compounds in the solution. The distillation bench evaporates individual materials by using the different boiling points of the elements in the initial vessel, and by using that, the agent is able to isolate materials in their own vessels. Each bench utilizes vessels to contain materials and perform processes on the vessels. The reaction bench uses a single vessel while the extraction and distillation benches each use three vessels to ease the separation process. On top of that, there is a lab manager that is able to direct and inspect results from sub agents that carry out tasks for individual benches, and is also able to take materials from bench to bench using the vessels. Each bench can be initiated with an empty vessel(s) or be provided with a vessel(s) from previously completed benches using the lab manager. Make sure to check out our documentation here for more information about the Lab Manager or to learn more about the benches and what they have to offer!
